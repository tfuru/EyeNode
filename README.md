# EyeNode
このプロジェクトは AndroidでUSBカメラを使って 同じネットワーク内のローカルLLM で画像解析し、結果を表示・読み上げるAIアシスタントアプリです。

## 主な機能 (Features)
- **USBカメラ連携**: Android端末に接続したUSBカメラからの映像を取得します。
- **指差しトリガー (Finger Trigger)**: MediaPipe Hands を使用し、カメラ映像内での指差しジェスチャーをリアルタイムで検知。指（人差し指）を立てたタイミングで自動的に解析を開始します。
- **電話ジェスチャー対話トリガー (Phone Call Dialogue)**: 親指と小指を立てるジェスチャーを検知すると、即座に音声認識を開始。キーワードなしでAIと直接対話（質問など）が可能です。
- **音声トリガー (Voice Trigger)**: 指定したキーワード（「これ何？」など）に反応して、ハンズフリーで解析を開始できます。
- **自動解析モード (Auto Analysis)**: 設定した間隔で自動的に解析を繰り返すモード。常に周囲を見守るアシスタントとして動作します。
- **ローカルLLMによる画像解析**: 同じローカルネットワーク上に配置されたLLMサーバーへ画像を送信し、AIによる解析を実行します。
- **音声合成 (Speech Synthesis)**: VOICEVOX APIと連携し、解析結果をリアルタイムで読み上げます。複数のキャラクター（話者スタイル）から選択可能です。
- **解析画像のサムネイル表示**: 直近の解析対象となった画像を画面端に表示。タップすることで全画面で拡大確認ができます。

## アーキテクチャ構成 (Architecture)
- **クライアント**: Androidアプリケーション (Jetpack Compose / MediaPipe)
- **サーバー**: 
  - LLMサーバー（例: Ollama, vLLM, llama.cpp など、OpenAI互換APIを想定）
  - 音声合成サーバー（VOICEVOX API 拡張版）
- **ハードウェア**: Android端末、USBカメラ（UVC対応）

## 動作環境 (Prerequisites)
- Android OSが搭載された端末 (USB OTG対応)
- UVC (USB Video Class) 対応のUSBカメラ
- ローカルLLM環境 (同じWi-Fi/ローカルネットワーク内で稼働していること)
- VOICEVOXサーバー（音声読み上げを使用する場合）
